Class FormGroupDirective

description

Binds an existing FormGroup to a DOM element.

This directive accepts an existing FormGroup instance. It will then use this FormGroup instance to match any child FormControl, FormGroup, and FormArray instances to child FormControlName, FormGroupName, and FormArrayName directives.

Set value: You can set the form's initial value when instantiating the FormGroup, or you can set it programmatically later using the FormGroup's {@link AbstractControl#setValue setValue} or {@link AbstractControl#patchValue patchValue} methods.

Listen to value: If you want to listen to changes in the value of the form, you can subscribe to the FormGroup's {@link AbstractControl#valueChanges valueChanges} event. You can also listen to its {@link AbstractControl#statusChanges statusChanges} event to be notified when the validation status is re-calculated.

Furthermore, you can listen to the directive's ngSubmit event to be notified when the user has triggered a form submission. The ngSubmit event will be emitted with the original form submission event.
