I18nExpInstruction

I18nExpInstruction: number | string
  • Defined in node_modules/@angular/core/src/render3/i18n.d.ts:37

Represents the instructions used to translate attributes containing expressions. Even indexes contain static strings, while odd indexes contain the index of the expression whose value will be concatenated into the final translation.

I18nInstruction

I18nInstruction: number | string
  • Defined in node_modules/@angular/core/src/render3/i18n.d.ts:31

Represents the instructions used to translate the template. Instructions can be a placeholder index, a static text or a simple bit field (I18nFlag). When the instruction is the flag Text, it is always followed by its text value.

i18nMapping

  • i18nMapping(translation: string, elements: object[] | null, expressions?: object[] | null, templateRoots?: string[] | null, lastChildIndex?: number | null): I18nInstruction[][]
    • Defined in node_modules/@angular/core/src/render3/i18n.d.ts:63

    Takes a translation string, the initial list of placeholders (elements and expressions) and the indexes of their corresponding expression nodes to return a list of instructions for each template function.

    Because embedded templates have different indexes for each placeholder, each parameter (except the translation) is an array, where each value corresponds to a different template, by order of appearance.

    Parameters

    • translation: string

      A translation string where placeholders are represented by {$name}

    • elements: object[] | null

      An array containing, for each template, the maps of element placeholders and their indexes.

    • Optional expressions: object[] | null

      An array containing, for each template, the maps of expression placeholders and their indexes.

    • Optional templateRoots: string[] | null

      An array of template roots whose content should be ignored when generating the instructions for their parent template.

    • Optional lastChildIndex: number | null

      The index of the last child of the i18n node. Used when the i18n block is an ng-container.

    Returns I18nInstruction[][]

    A list of instructions used to translate each template.
