whenRendered

  • whenRendered(component: any): Promise<null>
    • Defined in node_modules/@angular/core/src/render3/component.d.ts:120

    Wait on component until it is rendered.

    This function returns a Promise which is resolved when the component's change detection is executed. This is determined by finding the scheduler associated with the component's render tree and waiting until the scheduler flushes. If nothing is scheduled, the function returns a resolved promise.

    Example:

    await whenRendered(myComponent);

    Parameters

    • component: any

      Component to wait upon

    Returns Promise<null>

    Promise which resolves when the component is rendered.
