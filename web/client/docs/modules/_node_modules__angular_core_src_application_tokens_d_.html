Const APP_ID

APP_ID: InjectionToken<string>
  • Defined in node_modules/@angular/core/src/application_tokens.d.ts:20

A DI Token representing a unique string id assigned to the application by Angular and used primarily for prefixing application attributes and CSS styles when {@link ViewEncapsulation#Emulated ViewEncapsulation.Emulated} is being used.

If you need to avoid randomly generated value to be used as an application id, you can provide a custom value via a DI provider configuring the root Injector using this token.
