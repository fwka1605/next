chord

  • chord(...var_args: Array<string | IKey>): string
    • Defined in node_modules/@types/selenium-webdriver/index.d.ts:1586

    Simulate pressing many keys at once in a 'chord'. Takes a sequence of keys or strings, appends each of the values to a string, and adds the chord termination key ({@link Key.NULL}) and returns the resulting string.

    Note: when the low-level webdriver key handlers see Keys.NULL, active modifier keys (CTRL/ALT/SHIFT/etc) release via a keyup event.

    Parameters

    • Rest ...var_args: Array<string | IKey>

      The key sequence to concatenate.

    Returns string

    The null-terminated key sequence.
