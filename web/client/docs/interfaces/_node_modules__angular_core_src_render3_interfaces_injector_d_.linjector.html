bf0

bf0: number
  • Defined in node_modules/@angular/core/src/render3/interfaces/injector.d.ts:42

The following bloom filter determines whether a directive is available on the associated node or not. This prevents us from searching the directives array at this level unless it's probable the directive is in it.

  • bf0: Check directive IDs 0-31 (IDs are % 128)
  • bf1: Check directive IDs 32-63
  • bf2: Check directive IDs 64-95
  • bf3: Check directive IDs 96-127
  • bf4: Check directive IDs 128-159
  • bf5: Check directive IDs 160 - 191
  • bf6: Check directive IDs 192 - 223
  • bf7: Check directive IDs 224 - 255

See: https://en.wikipedia.org/wiki/Bloom_filter for more about bloom filters.

cbf0

cbf0: number
  • Defined in node_modules/@angular/core/src/render3/interfaces/injector.d.ts:56

cbf0 - cbf7 properties determine whether a directive is available through a parent injector. They refer to the merged values of parent bloom filters. This allows us to skip looking up the chain unless it's probable that directive exists up the chain.
